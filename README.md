# Selenium WebDriver with Java

This repository contains my practice work while taking the Udemy course "Selenium WebDriver with Real Time Examples + Interview Questions."  
The goal of this repository is to learn and practice web automation using Selenium and Java through practical examples.

## Course

Course link:  
https://www.udemy.com/course/selenium-real-time-examplesinterview-questions/

The course focuses on practical Selenium WebDriver usage, automation concepts, and interview-related examples.

- Selenium WebDriver
- TestNG
- Maven
- Git

Examples and exercises in this repository may include:

- Selenium WebDriver setup
- Locating web elements
